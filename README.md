# Installation

```bash
virtualenv venv
```

Then, activate the environment:

```bash
source venv/bin/activate
```

Install required packages

```bash
pip install -r requirements.txt
```

## Running

```bash
python manage.py runserver
```

## Additional - ORM
```bash
python manage.py migrate
```
